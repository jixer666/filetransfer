# 文件流转系统

## 系统简介

文件流转系统是一个基于Web的文件共享平台，支持用户注册、文件上传下载、管理员审核等功能。系统采用本地文件存储方式，确保数据安全可控。

## 系统功能模块

### 1. 用户认证系统

#### 用户类型
- **普通用户**：文件上传下载功能
- **管理员用户**：用户管理、数据管理功能
- **访客用户**：仅文件下载功能

#### 注册流程
- 普通用户注册入口："普通用户注册"
- 注册必填信息：
  - 用户名
  - 密码
  - 本级主要单位
  - 下级所属单位
  - 真实姓名
  - 固定电话
  - OA办公账号
- 新用户需管理员审核通过后才能登录

#### 登录验证
- 被停用用户登录时显示："账户已停用，请联系管理员"

### 2. 普通用户功能

#### 文件上传
- 支持单个文件或压缩包上传
- 文件大小限制：≤ 2GB (2048MB)
- 实时显示上传进度
- 支持断点续传（网络中断后可恢复）
- 自动生成：
  - **提取码**：5位字母数字组合，用于文件下载
  - **删除码**：用于文件删除
- 文件有效期：48小时自动删除

#### 文件管理
- 历史记录查看：
  - 上传/下载记录
  - 文件状态显示：有效/已删除/已过期
- 个人中心：
  - 查看和修改个人信息
  - 用户名、用户角色、单位信息等

### 3. 管理员功能

#### 用户管理板块

**1. 新用户审核**
- 查看待审核用户注册申请
- 审核信息：用户名、单位信息、联系方式等
- 审核状态：待处理、已同意、已拒绝
- 排序规则：待处理置顶 + 注册时间降序

**2. 用户管理**
- 查看所有注册用户信息
- 字段包括：用户名、角色、单位信息、注册时间、上传下载次数等
- 账户状态管理：正常/禁用
- 支持用户停用、删除操作
- 级联搜索功能

#### 数据管理板块
- 查看所有上传文件信息
- 字段包括：上传用户、文件名、文件类型、提取码、上传时间等
- 文件状态：有效/已删除/已过期
- 支持文件信息删除操作
- 级联搜索功能

### 4. 访客模式
- 无需登录即可访问
- 仅提供文件下载功能
- 通过提取码下载文件

## 技术特性

### 文件存储
- 文件存储在本地磁盘
- 数据库仅存储文件路径信息
- 文件存储路径可配置修改

### 系统要求
- 支持大文件上传（最大2GB）
- 断点续传功能
- 实时进度显示

## 使用说明

### 普通用户操作流程
1. 注册账号 → 等待管理员审核 → 登录系统
2. 上传文件 → 获取提取码和删除码
3. 分享提取码给其他用户下载
4. 在历史记录中查看文件状态

### 管理员操作流程
1. 登录管理员账户
2. 在"新用户审核"中审批注册申请
3. 在"用户管理"中管理用户账户
4. 在"数据管理"中监控文件流转情况

### 访客操作流程
1. 进入访客模式页面
2. 输入提取码下载文件

## 注意事项

1. **账户安全**：妥善保管删除码，避免文件被误删
2. **文件时效**：上传的文件48小时后自动删除
3. **网络稳定性**：大文件上传建议保持网络稳定
4. **账户状态**：被停用用户无法登录系统
